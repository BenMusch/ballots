Ballot Configuration

How to Print

For best results print from Chrome. In other browsers margins and layouts may be off.

Ensure "Print backgrounds" is on and "Print headers and footers" is off. Use your browser's print function (File > Print) and set paper size/orientations as desired. In the "Show Details" section set "Pages" to be from 1 to 1 and set "Print Background Colors" on. If possible reduce the default "Margins". Use your browser's print function (File > Print) and set paper size/orientations as desired. In the "More Settings" section ensure "Simplify Page" is off and "Background graphics" is on. You may also want to reduce the default "Margins".
